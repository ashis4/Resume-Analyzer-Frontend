# 🎨 AI Resume Analyzer – Frontend

## 📌 Overview

The **AI Resume Analyzer Frontend** is a responsive web interface that allows users to upload their resumes and receive AI-powered analysis. It interacts with the backend API to provide ATS scores, skill insights, and improvement suggestions in a clean and user-friendly way.

This frontend is designed to deliver a smooth user experience with modern UI components and real-time feedback.

## ⚙️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & layout
* **JavaScript (Vanilla JS)** – Functionality & API integration
* **Netlify** – Deployment

---

## ✨ Features

* 📄 Upload resume (PDF/DOC)
* 🔗 Connects with backend API for analysis
* 📊 Displays ATS score and feedback
* 🧠 Shows AI-generated suggestions
* 🎯 Clean and responsive UI design
* ⚡ Fast and lightweight performance

---

## 🖥️ How It Works

1. User uploads a resume
2. Frontend sends file to backend API
3. Backend processes it using AI (Gemini)
4. Results are displayed with score and suggestions
